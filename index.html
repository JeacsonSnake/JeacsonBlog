TypeScript

TypeScript 是 JavaScript 的超集，添加了静态类型系统和面向对象特性，广泛用于大型前端项目。

与 JavaScript 的关系

  • TypeScript 代码需编译为 JavaScript 运行
  • 所有 JavaScript 代码都是合法的 TypeScript 代码（可选类型标注）
  • Vue3 全面采用 TypeScript 重写

核心特性

  • 静态类型（编译时检查）
  • 接口（Interface）与类型别名（Type Alias）
  • 泛型（Generics）
  • 装饰器（Decorators）

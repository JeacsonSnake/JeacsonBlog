## 简述

这里是我有关ROS2的相关笔记，这些笔记内所使用的相关软硬件配置为：

- ROS2 Jazzy Jalisco LTS
- Ubuntu 24.04.1 LTS
- Raspberry Pi 5B (4GB)

- ROS2-Jazzy官方支持文档: [ROS 2 Documentation — ROS 2 Documentation: Jazzy documentation](https://docs.ros.org/en/jazzy/)

# 分布式通信

ROS2 基于 DDS（Data Distribution Service）实现分布式通信，同一局域网内的不同主机可以通过配置 ROS_DOMAIN_ID 实现节点发现和数据交互。

## 域ID机制

- 默认域ID = 0
- 不同域ID 的节点互相不可见
- 通过设置环境变量 `ROS_DOMAIN_ID` 改变域

## 与话题通信的关系

分布式通信本质上是将话题通信扩展到网络层面，仍然遵循发布/订阅模式。

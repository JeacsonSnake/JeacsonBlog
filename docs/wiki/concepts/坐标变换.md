# 坐标变换 (Coordinate Transform)

坐标变换是机器人学中的核心概念，ROS2 通过 tf2 库实现。

## 核心消息类型

- **TransformStamped** — 描述两个坐标系之间的变换关系（平移+旋转）
- **PointStamped** — 带时间戳的点坐标

## 通信双方

- **广播方 (Broadcaster)** — 发布坐标变换关系
- **监听方 (Listener)** — 订阅并使用坐标变换

## 典型应用

- 机器人底盘坐标系与传感器坐标系之间的变换
- 地图坐标系与机器人坐标系的绑定
- 多传感器融合

Web 开发

Web 开发是互联网应用构建的实践领域，涵盖前端、后端、网络协议等多个方面。

技术栈

前端

  • HTML / CSS / JavaScript
  • 框架：[[../entities/Vue.md]]、React、Angular
  • 构建工具：Vite、Webpack

通信协议

  • HTTP/HTTPS — Web 基础协议
  • DNS — 域名解析
  • TCP/IP — 传输控制协议
  • 参考：[[../entities/Computer_Network.md]]

JavaScript 生态

  • [[../entities/JavaScript.md]]
  • [[../entities/TypeScript.md]]

关键流程

输入 URL 回车后发生了什么

  1. DNS 解析 → 域名 → IP
  2. TCP 三次握手建立连接
  3. HTTP 请求/响应
  4. 浏览器渲染

JavaScript

JavaScript 是一门多范式编程语言，是 Web 开发的三大核心技术之一（HTML + CSS + JavaScript）。

基本信息

  • 类型：高级、动态、解释型语言
  • 数据类型：7 种原始类型（undefined, null, boolean, string, number, symbol, bigint）+ object
  • 运行时：浏览器、V8 引擎、Node.js
